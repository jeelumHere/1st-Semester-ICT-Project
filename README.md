# Culture of Pakistan

## Overview
**Culture of Pakistan** is an ICT-based web project developed using **HTML, CSS, and JavaScript**.  
The project aims to showcase the rich cultural heritage of Pakistan through a structured, multi-page website with interactive elements.

## Website Structure

### Homepage
- The homepage is based on the theme **“Culture of Pakistan”**
- It visually represents Pakistan’s cultural diversity
- A navigation bar allows users to explore different sections of the website

### Cultural Sections
The website includes **six main web pages**, each dedicated to an important aspect of Pakistani culture:
- **Poetry**
- **Art**
- **Cuisines**
- **Dresses**
- **Sufism**
- **Festivals**

Each page provides relevant information related to its topic.

### Navigation Bar & Dropdowns
- The navigation bar is accessible from the homepage
- Each menu item contains a **dropdown menu** linking to content related to that section
- A **Quiz** link is also included in the navigation bar

### Quiz Page
- An additional quiz page is included
- The quiz allows users to test their knowledge about Pakistani culture

### Footer
- A footer is present on every web page
- The footer contains:
  - Project description
  - Group member details
  - Other relevant project information

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Purpose
This project was developed as part of an **ICT course** to strengthen skills in:
- Front-end web development
- Website navigation and structure
- Styling using CSS
- Basic interactivity using JavaScript
